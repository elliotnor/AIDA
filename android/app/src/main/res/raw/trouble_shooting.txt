
If there are issues with connecting to AIDA, the problems are best investigated by following these steps:

-Restart AIDA

-Are you trying to connect to the correct IP address and port?

-Make sure the IP address is set to 192.168.50.195

-Make sure the port is set to 6662

-Ensure that the AIDA network is up and running

-Ensure that WIFI is enabled on the tablet
